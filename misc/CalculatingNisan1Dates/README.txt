##############################################################################
The scripts in this directory create CSV files with planet information related
to the Nisan 1 date (first new moon of each year from a Hebrew calendar
perspective).

These dates are caledulated in various potential ways to test and determine
what actually works for producing results.

##############################################################################

Usage:

1) Open the following script:

calculateNisan1Dates.py

2) Update variables
     - ephemeris location
     - start and end timestamps
     - desired method for Nisan 1 calculation.

3) Run the script:

    python3 calculateNisan1Dates.py


This should produce the following files, respectively:

    nisan1Dates.csv
