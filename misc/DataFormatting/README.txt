
Steps to get data:

 - Open and edit getFuturesData_PriceData.bash to change variable
   settings for the desired futures symbol.

 - Run: getFuturesData_PriceData.bash

 - Open and edit getFuturesData_TFCCommodityCharts.bash to change
   variable settings for the desired futures symbol.

 - Run: getFuturesData_TFCCommodityCharts.bash



Note:

  Soybeans and some symbols may be different across provider.  

  Example: PriceData uses 'S' for CBOT pit soybeans and TFC uses 'SB'
  for CBOT pit soybeans.
