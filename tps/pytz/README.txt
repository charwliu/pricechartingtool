# pytz-2010h
##############################################################################

Description:

pytz is used to obtain timezone information.

The original library is only available for Python 2.6.  I took the
original library, ran 2to3 on it, and applies some fixes for it to
work on Python 3.1.

##############################################################################

To install:

tar -xjvf pytz-2010h.modsForPython3.tar.bz2
cd pytz-2010h
su
python3 setup.py install
